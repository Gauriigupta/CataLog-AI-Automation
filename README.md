# Product Catalog

A lightweight, responsive product catalog system designed to display and manage product listings efficiently. It includes search, filters, admin controls, and CSV support, making it suitable for e-commerce and inventory management.

## Features
- Real-time product browsing with category filters and pagination  
- Search by product name, tags, or SKU  
- Product detail pages with images, price, and specifications  
- Admin panel to add, edit, and delete products  
- CSV import/export for bulk management  
- Fully mobile-responsive UI

## Tech Stack
- **Frontend:** React / Next.js  
- **Backend:** Node.js + Express (optional)  
- **Database:** MongoDB or PostgreSQL  
- **Storage:** Local uploads or Cloud (AWS S3)  
- **Authentication:** JWT (optional)
